# 🛒 Carrinho de Compras - Estilo Shopee

Projeto desenvolvido como desafio prático da DIO com o objetivo de simular um carrinho de compras inspirado na interface da Shopee.

---

## 📌 Objetivo do Projeto

Este projeto tem como finalidade praticar conceitos de desenvolvimento web, organização de código e versionamento utilizando Git e GitHub.

O sistema simula funcionalidades básicas de um carrinho de compras, permitindo:

- Adicionar produtos
- Remover produtos
- Alterar quantidade
- Calcular subtotal
- Calcular valor total da compra

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
